# Spinosaurus

An interactive website dedicated to **Spinosaurus**, one of the most fascinating dinosaurs ever discovered. The project combines scientific information with a modern interface, smooth animations, an image gallery, and an integrated music player to create a more immersive experience.

## Features

* Responsive design for desktop, tablet, and mobile.
* Interactive timeline of important discoveries.
* Image gallery with lightbox preview.
* Taxonomy section with different *Spinosaurus* species.
* Animated discovery map.
* Built-in music player with playlist, volume control, and track selection.
* Smooth animations and subtle visual effects throughout the site.

## Built With

* HTML5
* CSS3
* JavaScript (Vanilla)

Clone the repository:

Open `index.html` in your browser, or run it with a local server such as VS Code Live Server.
